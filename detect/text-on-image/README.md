# Ritchie Detect Text On Image Formula

## Premisses

- [Tesseract](https://github.com/tesseract-ocr/tessdoc) installed locally.

## Command

```bash
rit detect text-on-image
```

## Description

This formula allows to detect text on images / pictures.
